# MorseCodeTransmitterUsingLight
The application transforms a text given by the user in light Morse code. The interface uses a 
4x4 Matrix Keyboard with 10 digit buttons and 6 control buttons. To be able to transmit text as 
well as numbers, each character is encoded such that 2 symbols from the keypad are 
equivalent to a specific character (00=A, 01=B … 25=Z and *0=0, *1=1 … *9=9). As the 
message is introduced, it will be displayed on the LCD. Pressing the button labeled “D” will 
delete the last character on the screen and pressing the “#” button will erase all characters on 
the screen.  By pressing the transmit button (C), the transmission will begin. The duration of a 
dot is 0.5 seconds.
